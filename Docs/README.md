AI Models (Llama, Gemma, Mixtral)

The React frontend communicates with the FastAPI backend, which orchestrates requests to multiple AI models through the grok API. Responses from these models are then processed through the council evaluation stages before a final answer is returned to the user.

Project Structure

The repository is organized into the following main components:

backend/
Contains the FastAPI-based Python backend responsible for handling requests, managing the AI council workflow, and communicating with the AI models.

frontend/
Contains the React-based user interface that allows users to interact with the system, submit queries, upload files or screenshots, and view AI responses.

docs/
Contains project documentation, architecture diagrams, screenshots, and other supporting materials related to the system.
